## [0.8.1] - 2025-12-15
### Added
- Remote raw `payload.bin` extraction support
- HTTP cookies support for remote requests
- Built-in DNS resolver via `hickory_dns` for fully static **Linux** binaries
- Runtime DNS override using `PAYLOAD_DUMPER_CUSTOM_DNS`
- Added C and C++ APIs for library usage

### Changed
- Minor internal improvements and refactoring
